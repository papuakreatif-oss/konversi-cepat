# 🔄 KonversiCepat — Universal Converter

**KonversiCepat** adalah aplikasi konversi universal berbasis web, dibangun dengan HTML/CSS/JS tanpa dependency eksternal. Diciptakan oleh Papua Kreatif Group.

## 🚀 Live Demo

🌐 **[github.com/papuakreatif-oss/konversi-cepat](https://github.com/papuakreatif-oss/konversi-cepat)**

## ✨ 11 Kategori Konversi

| # | Kategori | Unit |
|---|----------|------|
| 💰 | **Mata Uang** | IDR, USD, EUR, JPY, SGD, MYR, AUD, GBP, CNY, THB |
| ⚖️ | **Berat** | kg, g, mg, ton, lb, oz |
| 📏 | **Jarak** | km, m, cm, mm, mi, ft, in, yd |
| 🌡️ | **Suhu** | °C, °F, K |
| 🥤 | **Volume** | L, mL, gal_us, gal_uk, cup, m³ |
| 🏠 | **Luas** | m², km², ha, are, ft², acre |
| 🏎️ | **Kecepatan** | km/h, m/s, mph, knot |
| ⏰ | **Waktu** | hari, jam, menit, detik, minggu, bulan, tahun |
| ⚡ | **Energi** | J, cal, kcal, kWh, eV |
| 📊 | **Tekanan** | Pa, bar, psi, atm, mmHg |
| 💾 | **Data** | B, KB, MB, GB, TB, bit, Kbit |

## 🎨 Fitur UX

- **Real-time conversion** — ketik angka, langsung convert
- **Swap units** — tombol ⇅ tukar input/output
- **Quick convert** — 4 referensi cepat per kategori
- **Responsive** — mobile + desktop
- **Offline** — zero dependency, pure HTML+CSS+JS
- **Smart formatting** — exponential untuk angka besar/kecil, ID locale

## 🛠️ Penggunaan

```bash
git clone https://github.com/papuakreatif-oss/konversi-cepat.git
cd konversi-cepat
# Buka index.html di browser — tidak perlu server
```

## 📝 Catatan

- Currency rates **hardcoded** (perlu update manual untuk kurs terkini)
- Bisa live API kalau mau (e.g. exchangerate.host, frankfurter.app)
